## Installation

1. Add the repository to your `pom.xml`:
```xml
<repository>
    <id>tassia-nexus</id>
    <url>https://nexus.tassia.net/repository/maven-public/</url>
</repository>
```

2. Add the dependency to your `pom.xml`:
```xml
<dependency>
    <groupId>net.proxzi</groupId>
    <artifactId>core</artifactId>
    <version>1.1.0</version>
</dependency>
```
